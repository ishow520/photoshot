账号注册：
一、aws  直接去 咸鱼
重点：个人中心（网站右上角）——》 访问密钥 ---》 创建秘钥 复制出来
S3_UPLOAD_KEY + S3_UPLOAD_SECRET

左上角 s3
搜索S3
新建一个 S3
重点：开放权限

点进创建的s3--> 权限

屏蔽公共访问权限（存储桶设置） -- 全部取消勾选
存储桶策略
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "readonly policy",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "*",
            "Resource": "arn:aws:s3:::xxx/*"
        }
    ]
}
还需要在s3 里面的跨域中添加  http://localhist

二、SECRET
注册一个账号，然后点击   开发人员  复制 私钥 到配置文件
NEXT_PUBLIC_STRIPE_STUDIO_PRICE=50  最低50，不然支付不允许

三、邮箱，本地docker启动后
EMAIL_SERVER=smtp://localhost:25

本地发邮件之后 访问 http://localhost:1080/#/ 获取激活地址

四、REPLICATE
